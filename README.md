Python Project – Exploratory Data Analysis on Diwali Sales

Performed data cleaning and preprocessing on raw Diwali sales dataset using NumPy and Pandas.

Conducted Exploratory Data Analysis (EDA) to identify customer purchasing patterns, high-performing product categories, and revenue trends.

Used Matplotlib and Seaborn to create visualizations for better business insights.

Analyzed customer demographics to suggest strategies for improving customer experience and increasing revenue.

Provided data-driven conclusions to support decision-making and business growth.
